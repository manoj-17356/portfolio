Network Intrusion Detection System

Problem: Network systems face constant cyber threats, and traditional rule-based detection methods struggle to identify novel attack patterns.

Approach: Built a machine learning-based Intrusion Detection System (IDS) using the CFS-BA (Correlation-based Feature Selection with Bat Algorithm) technique to select optimal features from Microsoft telemetry data. Trained and compared LightGBM and Random Forest classifiers.

Result: Achieved 95.6% detection accuracy. Validated using confusion matrix analysis and ROC curve evaluation.
Tools: Python, Scikit-learn, LightGBM, Pandas, Matplotlib

Resume Screening using Machine Learning

Problem: Manual screening of large volumes of resumes is time-consuming and inconsistent, especially when matching candidates to specific job categories.

Approach: Built an automated system using Python and NLP techniques. Trained a K-Nearest Neighbors (KNN) classifier for multi-class job category prediction.

Result: Successfully classified resumes into relevant job categories with good accuracy across multiple classes.
Tools: Python, NLTK, Scikit-learn, Pandas, Matplotlib
